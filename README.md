### 発展版の技術選定と今後の展望

1. **フロントエンドのモダン化**
   - **Next.js (React) + TypeScript**
     を採用し、コンポーネント駆動での堅牢なUI構築と、SPAとしての快適なユーザー体験を実現します。

2. **バックエンド & AI対戦機能**
   - 無料のAI APIを活用した「AI対戦機能」の実装を予定しています。
   - AIからのレスポンスを特定のスキーマ形式（JSON）に厳密に制限するため、**Python
     (FastAPI + Pydantic)** での構築を第一候補としています。
   - ※ ただし、AIのレスポンス制御（Structured
     Outputs等）が同様に実現可能であれば、軽量な **Go** や **Ruby**
     でのバックエンド構築も視野に入れ、要件に合わせて柔軟に選定します。

3. **データベースの導入**
   - ユーザーごとの管理および「しりとりの対戦履歴」を永続化するため、リレーショナルデータベース（**PostgreSQL**
     を予定）を導入します。
